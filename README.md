# intercom-rank

### project purpose
intercom + alexa integration

### stack
Python 3, Flask, PostgreSQL, SQLAlchemy

### how to develop
- git clone
- mkvirtualenv --python=/usr/bin/python3 intercom-rank
- pip install -r requirements.txt
- ./run.py
- go to localhost:5000
